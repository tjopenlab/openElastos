Title:   ETSI PUBLICATIONS CATALOG download files
Files:   catcsv.exe and catcsv.lzh
Date:    29/03/2000 11:00:30 PM

catcsv.exe and catcsv.lzh contain a comma-delimited version of the ETSI PUBLICATIONS CATALOG.

The format of the delimited file is as follows:

 - each field is separated by a comma.
 - each text field is surrounded by double quotes.

Each Compressed File contains:

1. readme.txt: This text

2. catcsv.csv: The comma delimited file containing the data for database importing.

3. catcsv.lin: A comma delimited file containing the field definitions without explanation, this is for database importing.

4. catcsv.rec: List of field names along with their data-types.
